# git 사용법

- git --version
  - 설치된 깃 버전 확인
- git init
  - 현재 켜져있는 Project를 버전관리하겠다

* git add .
* 수정내용 저장하기
* git commit -m '버전 이름' \*버전이름의 버전 생성하기

---

# GitHub 연결하기

- git remote add origin 주소
  - origin이라는 별칭으로 원격 저장소를 연결

---

# GitHub에 업로드 하기

- git push origin master
  - origin이라는 별칭의 원격저장소에 버전 내역 전송
